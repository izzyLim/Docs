1. 전체 시스템 개요

BIP-Agents는 BIP-Pipeline(데이터 파이프라인)과 연동하여 시장 분석, 체크리스트 모니터링, 종목 추천을 수행하는 AI 에이전트 시스템입니다.

아키텍처 패턴: Airflow(오케스트레이션) → BIP-Agents API(인텔리전스) → 텔레그램/이메일(발송)

12. 폐기된 에이전트

모닝리포트 LangGraph 멀티 에이전트

DAG: morning_pulse_langgraph (paused)

구조: 4개 병렬 에이전트 (글로벌/한국/반도체/수급) → 어그리게이터 → 품질 체커

폐기 이유: 단일 LLM(llm_analyzer_v2)이 동일한 데이터로 충분한 품질의 분석을 생성. 멀티 에이전트 오버헤드 대비 개선이 미미하고 비용만 증가.

코드 유지: 삭제하지 않고 참조용으로 보관 (병렬 에이전트 패턴 참고 가능)

체크리스트 4단계 파이프라인 (Phase 2~4)

파일: checklist_parser.py, checklist_collector.py, checklist_signal.py, checklist_rules.py, checklist_explainer.py, checklist_graph.py

폐기 이유: 키워드 기반 파서가 LLM 출력 형식 변화에 취약. Phase 5에서 Haiku ReAct로 전환.

코드 유지: 삭제하지 않고 참조용으로 보관 (룰 엔진/도구 매핑 패턴 참고 가능)
