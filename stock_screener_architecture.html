1. 시스템 개요

목표: 한국 주식(KOSPI/KOSDAQ) 대상으로 매일 자동 종목 스크리닝 + Bull/Bear 토론 기반 투자 의견 제시

핵심 원칙: 스코어링/판정은 deterministic, LLM은 분석 메모 + 방향 판정만

참고: TradingAgents의 Bull/Bear debate discipline 차용 (10개 에이전트 전면 복제 아님)

매일 자동 실행

Airflow DAG → BIP-Agents API → 텔레그램 발송

개별 종목 분석

API 호출로 특정 종목 심층 분석 가능

2. 아키텍처 흐름

Phase 1: Deterministic 스크리닝 analytics_stock_daily + consensus_estimates + analytics_valuation → 149개 기본 후보 (시총 5000억+, 거래대금 10억+, 당일급등 제외) → 7개 프리셋 필터 → 30~40개 → 4카테고리 스코어링 + 품질 필터 (수익률 10%+, RR 1.5+) → 상위 15개 │ ▼ Phase 2: 멀티 에이전트 토론 (상위 5개, 병렬 실행) ┌─ 기술 분석가 (Haiku + MCP) ├─ 재무 분석가 (Haiku + MCP) ─── 4개 병렬 ──→ Analyst Memo ├─ 뉴스 분석가 (Haiku + MCP) └─ 수급 분석가 (Haiku + MCP) │ ┌─ Bull Researcher (Sonnet) ─── 매수 논거 └─ Bear Researcher (Sonnet) ─── 매도 논거 + 한국 특화 반증 │ Research Manager (Haiku) ─── 방향 판정 (bull_strong ~ bear_strong) │ Deterministic 등급: Phase1 스코어 + 방향 modifier → Buy / Overweight / Hold / Underweight / Sell │ ▼ stock_recommendations DB + 텔레그램 발송

3. Phase 1: Deterministic 스크리닝

SQL 기반 1차 필터

  • 데이터 소스: analytics_stock_daily + stock_info + analytics_valuation + consensus_estimates
  • 유동성 필터: 시총 5,000억+, 거래대금 10억+
  • 추격매수 방지: 당일 +5% 초과 급등주 제외
  • 수급 데이터: 최근 5일 외국인/기관 연속 순매수일수, 누적금액
  • 컨센서스: consensus_estimates 별도 JOIN (목표가, 투자의견)

품질 필터 (scorer 단계)

필터기준목적
최소 수익률목표가 / 현재가 ≥ 1.10 (+10%)수익성 없는 추천 배제
최소 손익비RR ≥ 1.50위험 대비 수익 불리한 종목 배제
진입가 > 손절가entry_high > stop_loss비정상 케이스 배제

목표가 계산 우선순위

  1. 컨센서스 목표가 (consensus_estimates.target_price) — 현재가 대비 +10% 이상일 때
  2. 52주 고점 복귀 — pct_from_52w_high에서 역산
  3. 피보나치 확장 — MA swing range × 1.618
  4. Fallback — 현재가 × 1.10 (최소 수익률)

4. Phase 2: 멀티 에이전트 토론

4개 Analyst Memo (병렬, Haiku)

각 에이전트가 MCP 도구로 실제 데이터를 수집하고 200자 이내 분석 메모 작성. 4개가 asyncio.gather로 동시 실행.

에이전트역할결론
기술 분석가추세, RSI/MACD, 지지/저항, 패턴긍정/부정/중립
재무 분석가매출/영업이익 추이, PER/PBR, DART 공시긍정/부정/중립
뉴스 분석가최근 뉴스, DART 공시, 업종 이슈긍정/부정/중립
수급 분석가외국인/기관 흐름, 프로그램 매매, 섹터 로테이션긍정/부정/중립

Bull Researcher (Sonnet)

  • 4개 analyst memo + Phase 1 스코어 입력
  • 매수 논거 작성 (200자)
  • 성장 잠재력, 기술적 반등, 수급 개선, 밸류 매력
  • 확신도 표시: 높음/중간/낮음

Bear Researcher (Sonnet)

  • 매도 논거 + 한국 특화 반증 강제
  • 대주주 리스크, CB/BW/유상증자
  • 잠정실적 쇼크, DART 공시 리스크
  • 외국인 이탈, 프로그램 매도, 커버리지 부족

Research Manager (Haiku)

Bull vs Bear 방향만 판정. 등급은 코드가 결정.

방향의미modifier
bull_strongBull 압도적, Bear는 일반 리스크만+2
bull_winBull 우세, Bear는 결정적이지 않음+1
neutral양쪽 비슷0
bear_winBear 우세, 구체적 위험 제시-1
bear_strong치명적 결함 (재무조작, 상장폐지 등)-2

6. 7개 스크리닝 프리셋

#전략핵심 조건스타일
1과매도 반등RSI<45 + MACD양전환 + 수급 + 풀백존(BB≤0.65, 이격≤3%)단기
2골든크로스MA5>MA20 2% 이내 + vol>1.2 + 풀백존단기
3가치 모멘텀PER<15 + ROE>10 + 외국인3일+ + 이격-5~2 + 목표가+10%중기
4돌파52w고점-2~-8% + vol>1.5 + MA20위 + MA5*1.03 이내단기
5심층 가치PER<7 + PBR<0.8 + ROE>8 + MA20근접 + MA60지지중기
6추세 추종정배열(MA20>60>120) + 52w-3~-15% + 수급 + RSI40~72중기
7VCP 돌파완전정배열(MA5>20>60>120) + vol200%+ + 52w-2~-15% + 수급단기

프리셋은 OR 조건 — 하나라도 매칭되면 후보에 포함. 한 종목이 여러 프리셋에 매칭 가능.

7. 4카테고리 스코어링

기술적 (40점)

  • RSI 과매도: 0~10점
  • MACD 양전환: 0~10점
  • 볼린저밴드 위치: 0~5점
  • 추세 (MA 기반): 0~5점
  • 거래량 비율: 0~5점
  • 52주 위치: 0~5점

수급 (25점)

  • 외국인 순매수: 0~10점
  • 기관 순매수: 0~8점
  • 외국인+기관 동반: +4점
  • 5일 누적 순매수: 0~3점

밸류에이션 (20점)

  • PER 매력도: 0~6점
  • PBR 저평가: 0~4점
  • ROE 수준: 0~4점
  • 컨센서스 괴리율: 0~6점

매크로 (15점)

  • VIX 수준: ±5점
  • KOSPI 등락: ±3점
  • 원/달러 환율: ±2점
  • 중립 시작 (7.5점)
