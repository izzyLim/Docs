1. 시스템 개요

목표: 모닝리포트 체크리스트를 장 시작 전 / 장중 / 장 마감 시점에 실시간 데이터로 확인하고 텔레그램 알림 발송

핵심 원칙: LLM(Haiku)이 체크리스트를 읽고 필요한 MCP 도구를 직접 선택/호출. 키워드 파싱 불필요.

6. 프롬프트 설계

장 시작 전

"장 시작 전 항목만 확인. 한국 주식 실시간 조회 불가. EWY/환율/뉴스만 사용."

장중

"장전 항목 제외, 나머지 전부 확인. 주요 종목 현재가 포함."

장마감

"전체 항목 최종 결과. 종합점검 + 내일 관찰 포인트."

출력 형식 (모든 시간대 공통)

각 항목 2줄:
✅ *{항목 원문 요약}*
{신호등} {현재값 vs 기준 비교 1줄}

마지막:
———————————————
📊 *주요 종목*
삼성전자 xxx원 (+x.x%)
SK하이닉스 xxx원 (+x.x%)
→ 특이 움직임 1줄

💡 *종합*: 1줄

프롬프트 금지 사항

  • percentile, zscore, 90일 맥락 분석 — 불필요
  • 추가 섹션/해설/권장조치/테이블
  • ##헤딩, **볼드** (텔레그램은 *볼드*만)
  • 같은 indicator에 대한 2개 이상 항목
